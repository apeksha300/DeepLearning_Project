# Apple Leaf Disease Detection Using CNN

## Project Overview
This project uses Convolutional Neural Network (CNN) to detect diseases in apple leaves from images. It helps farmers and agriculture experts identify leaf diseases quickly and accurately.

## Features
- Upload leaf image
- Predict disease class
- Deep learning based model
- Fast and accurate detection

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- OpenCV
- Google Colab

## Dataset
PlantVillage Dataset containing apple leaf disease images.

## Disease Classes
- Apple Scab
- Black Rot
- Cedar Apple Rust
- Healthy Leaf

## Model Architecture
CNN with:
- Convolution Layers
- Max Pooling
- Flatten Layer
- Dense Layers
- Softmax Output

## Accuracy
Training Accuracy: 95%
Validation Accuracy: 93%

## How to Run
1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells
4. Upload image for prediction
